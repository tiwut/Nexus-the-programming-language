The Vision

Nexus is designed to bridge the gap between Python's simplicity and C++'s performance. With a refined set and fn syntax, it provides an intuitive way to build GUI apps, system tools, and network scripts.

How to Run Nexus

Running your first .nx file is easy. Follow these steps:

  1. Ensure Python 3.x is installed.
  2. Save the Nexus Interpreter as interpreter.py.
  3. Create a file named app.nx.
  4. Open your terminal and type:
python3 interpreter.py app.nx

Nexus Syntax

Clean, readable, and powerful.

# Create a variable
set greeting = "Hello Nexus World"

# Define a function
fn welcome_user()
    out greeting
    gui.msg("Welcome to the Glass Interface!")
end

# Logic and GUI
if OS == "Linux"
    welcome_user()
    gui.window("Nexus v0.6")
    gui.run()
end
